## 功能模块

### 1. 导航栏模块 (navbar.html + navbar.css)
- 独立的导航栏组件
- 包含Logo、品牌名称、用户头像和设置按钮
- 响应式设计，支持移动端

### 2. 主应用页面 (soul-match.html)
- **六个步骤的灵魂匹配流程：**
  1. **兴趣标签** - 选择用户兴趣爱好
  2. **性格测评** - 选择性格类型
  3. **上传性格报告文件** - 上传文件后，调用接口进行AI分析
  4. **星座选择** - 选择星座或自定义
  5. **生活理念** - 选择生活态度和价值观
  6. **社交偏好** - 选择社交方式和环境

- **用户信息管理：**
  - 点击右上角头像弹出用户信息弹窗
  - 支持编辑个人信息（姓名、性别、年龄、职位、地区、个人介绍）
  - 支持头像上传和预览
  - 自动弹出功能（新用户count=1时）

### 3. 登录页面 (login.html)
- 用户登录和注册功能
- 保存Authorization token和用户ID
- 支持新用户标识（count值）

## 技术特性

### 前端技术
- **HTML5** - 语义化标签和现代HTML特性
- **CSS3** - 渐变背景、动画效果、响应式设计
- **JavaScript ES6+** - 模块化代码、异步处理
- **GSAP** - 动画库，提供流畅的交互效果
- **Lenis** - 平滑滚动库

### 交互功能
- **动态加载** - 导航栏组件动态加载
- **表单验证** - 完整的表单验证机制
- **文件上传** - 支持图片上传和预览
- **动画效果** - 丰富的交互动画
- **响应式设计** - 适配各种屏幕尺寸

### 后端接口
- **用户信息获取** - `GET /information/{userId}`
- **用户信息更新** - `POST /upload`
- **认证机制** - Authorization token验证
